发表于 2017-05-20 | 分类于 start | 阅读次数

从2011年10月11号开始在北京品恩科技股份有限公司实习生开始自己的职业生涯，实习
转正 成为核心员工在到离职；从北京回南昌就职广东迪科思股份有限公司，江西尚通科
技有限公司，中安德信股份有限公司 再到自己创业（南昌在于科技有限公司）；时间跨
度还是比较大，中间也没有思考过从中技术上有学到什么,管理上有学到什么，人际关系
怎么样，创业有一段时间，接触了一些销售牛逼的人，一些技术上牛逼的人，又在网上
看了一些比较牛逼Bolg人物，古觉得自己现在有必要对自己之前的学习 工作及生活做
一个总结。下面以从业公司为单位总结一下。后期将把自己项目中一些技术能公开的就整
理出来，这也是对自己知识的一个梳理吧。而且以后会坚持写作。
